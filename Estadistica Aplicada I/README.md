Este proyecto trata el tema de Violencia Intrafamiliar en México durante el período de 2003 a 2014.

El proyecto consiste en el material que se encuentra dentro de las carpetas "Datos" y "Reportes" y de un archivo
adicional de tipo R Markdown.

La carpeta "Datos" contiene el diccionario de datos y dos carpetas: "BrutosCSV" y "BrutosCSV_funcionan". Ambas carpetas contienen
los datos que se utilizaron para el proyecto en versión PDF y CSV, pero sólo se utiliza la segunda para hacer los 
analisis, pues es la que minimiza los errores en el código.

La carpeta "Reporte" contiene el reporte técnico y el ejecutivo

El archivo R Markdowm, ProyectoMK.rmd, contiene el código que se escribió para hacer la lectura de los datos y para 
producir las gráficas presentadas en los reportes 

Se incluye un archivo adicional, ProyectoScript.R, en el que se presenta unicamente el código del proyecto y que se 
puede ejecutar directamente desde R con mayor facilidad.
